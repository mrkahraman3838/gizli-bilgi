Bilgisayarların Yapısı — Kısa ve Anlaşılır (Kodlu Sayfa)

Bu tek dosyalık HTML sayfası Visual Studio Code içinde açılmak üzere hazırlanmıştır. İçerik: Donanım, yazılım ve çalışma prensipleri.

1. Genel Bakış

Bilgisayarlar iki ana parçadan oluşur: Donanım (hardware) ve Yazılım (software). Bu sayfa, her bir bileşeni kısa ve görselle desteklenmiş şekilde açıklar.

2. Donanım (Hardware)

  • CPU (İşlemci): Hesaplamaları yapar; komut döngüsünü yönetir.
  • RAM (Bellek): Geçici veri depolama; çalışırken programlar burada tutulur.
  • Depolama (SSD/HDD): Kalıcı veri depolama.
  • Anakart: Parçaları birbirine bağlayan kart; veri yolu ve güç dağılımı sağlar.
  • G/Ç Birimleri: Klavye, fare, ekran, ağ kartı, USB cihazları gibi giriş-çıkış elemanları.

3. Yazılım (Software)

  • İşletim Sistemi (OS): Donanımı yönetir ve uygulama çalıştırma ortamı sağlar (Windows, Linux, macOS).
  • Sürücüler (Drivers): Donanım ile işletim sistemi arasındaki köprüdür.
  • Uygulamalar: Kullanıcıya hizmet eden yazılımlar (tarayıcı, ofis, oyunlar).

4. Çalışma Prensibi — Giriş → İşlem → Çıkış

Basit akış: kullanıcı bir komut verir (giriş), CPU komutu işler (işlem), sonuç görüntülenir veya saklanır (çıkış).

Akışı Görselleştir
1. Giriş
Klavye, fare, sensör veya ağ üzerinden veri.
2. Bellek & CPU
Veri RAM'e yüklenir → CPU talimatları okur ve işler.
3. Depolama & Çıkış
Sonuç kaydedilir veya ekranda gösterilir; çıkış cihazlarına gönderilir.

5. Örnek Kod — Basit CPU Döngüsü (Simülasyon)

Aşağıdaki JavaScript, çok basit bir komut işleme döngüsünü simüle eder. VS Code'da bir .html dosyasına yapıştırıp çalıştırabilirsiniz.

// Basit CPU döngüsü simülasyonu
const memory = [
  {op: 'LOAD', value: 5},
  {op: 'ADD', value: 10},
  {op: 'STORE', addr: 0},
  {op: 'HALT'}
];

let acc = 0; // accumulator
let pc = 0;  // program counter

function step() {
  const instr = memory[pc];
  if (!instr || instr.op === 'HALT') return false;
  switch(instr.op) {
    case 'LOAD': acc = instr.value; break;
    case 'ADD': acc += instr.value; break;
    case 'STORE': memory[instr.addr] = {value: acc}; break;
  }
  pc++;
  return true;
}

while(step()){}
console.log('Çalışma bitti. Accumulator =', acc);
Bu dosyayı Visual Studio Code içinde açıp düzenleyebilirsiniz. İsterseniz açıklamaları genişletip eğitim amaçlı slayt hâline dönüştürebilirim.
